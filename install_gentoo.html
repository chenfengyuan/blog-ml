2.1.1 btrfs multi device

#以后这里可以添加背景故事 参考文献: btrfs rootfs multi device: http://forums.gentoo.org/viewtopic-t-565360-postdays-0-postorder-asc-start-625.html 如何制作initramfs: http://en.gentoo-wiki.com/wiki/Initramfs

原理： 我也不明白。貌似是如果是多device的话，所有的device要active下啥的。 所以用/bin/btrfsctl -A /dev/sdxn来弄。 所以就要写个initramfs了。 步骤:

  1. 我借助genkernel生成的initramfs.修改init,在合适的地方执行/bin/btrfsctl -A xxx 我不想写死在initramfs，所以学习原脚本解析参数
    1. 在for x in ${CMDLINE}添加一个选择 btrfs\=*) USEBTRFSDEVICE=1 BTRFSDEVICES=`parseopt "${x}"` ;;
    2. 在 # Determine root device后面添加 if [ "${USEBTRFSDEVICE}" = 1 ] then bin/btrfsctl -A ${REALROOT} echo "${BTRFSDEVICES}"|sed 's,/\n/'|while read btrfs do /bin/btrfsctl -A "${btrfs}" done fi
  2. 编译一个静态的btrfsctl
    1. ebuild `equery which btrfs-progs` install
    2. cd /var/tmp/portage/sys-fs/btrfs-progs-0.19/work/btrfs-progs-0.19
    3. \rm btrfsctl
    4. make -n btrfsctl 应该可以看到一个命令，在其中添加-static，再执行
    5. gcc -static xxxxx
    6. ldd btrfsctl 可以看到已经是静态的了
  3. 照教程生成initramfs 重启选择即可。写得不好主要没时间了。如果有需要的话，我可以重新写下。其实参考资料里讲得很清楚。

also can use subvolid rootflags is the opition to mount root ro [KNL] Mount root device read-only on boot root= [KNL] Root filesystem rootdelay= [KNL] Delay (in seconds) to pause before attempting to mount the root filesystem rootflags= [KNL] Set root filesystem mount option string rootfstype= [KNL] Set root filesystem type rootwait [KNL] Wait (indefinitely) for root device to show up.   • automount:

    encfs /path/to/from /path/to/to

2.8 cpufreq

power managenment guide

  1. compile the kernel,use built-in,not module. CPU Frequency Scaling —> [*] CPU Frequency scaling [X] Enable CPUfreq debugging < > CPU frequency translation statistics [X] CPU frequency translation statistics details Default CPUFreq governor (userspace) <*> 'performance' governor <*> 'powersave' governor <*> 'ondemand' cpufreq policy governor <*> 'conservative' cpufreq governor <*> CPU frequency table helpers <M> ACPI Processor P-States driver <*> CPUFreq driver for your processor
  2. add modules="acpi-cpufreq" in /etc/conf.d/modules
  3. rc-update add cpufrequtils
  4. modify STARTOPTS="–governor conservative" in cpufrequtils
  5. /etc/init.d/cpufrequtils start

2.9 font

Howto install and use X Windows fonts

  1. Make a directory in usr/share/fonts or ~/.fonts/ Example: mkdir usr/share/fonts/freetype
  2. Copy fonts to directory
  3. Make indexes Creating fonts.dir Execute: mkfontdir usr/share/fonts/freetype
  4. Refresh font cache Execute: fc-cache Restart any programs you wish to use the new fonts
  5. add xft to USE
